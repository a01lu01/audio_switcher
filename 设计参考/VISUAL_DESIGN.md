# Batch File Renamer - 前端视觉设计参考

## 1. 设计风格概述

**风格定位**：Linear Design System 极简主义风格

**核心特征**：
- 大量留白，呼吸感强
- 扁平化设计，微阴影层次
- 圆角系统统一（6px / 8px / 12px）
- 冷色调为主，蓝色点缀
- 精致的过渡动画

## 11. 动画规范

### 11.1 过渡时长

```
┌─────────────────────────────────────────┐
│  duration-fast   150ms   微交互反馈     │
│  duration-normal 200ms   元素状态变化   │
│  duration-slow   300ms   区块展开/收起  │
└─────────────────────────────────────────┘
```

### 11.2 缓动函数

```
ease-out:    cubic-bezier(0.16, 1, 0.3, 1)   ← 元素进入
ease-in-out: cubic-bezier(0.4, 0, 0.2, 1)   ← 状态切换
```

### 11.3 关键动画

```css
/* 淡入上浮 */
@keyframes animate-in {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

/* 脉冲 */
@keyframes ping {
  75%, 100% { transform: scale(1.5); opacity: 0; }
}

/* 旋转加载 */
@keyframes spin {
  to { transform: rotate(360deg); }
}
```

## 14. 暗色模式适配

```
┌─────────────────────────────────────────────────────────────┐
│  亮色模式                    │  暗色模式                     │
│  ────────────────────────── │ ──────────────────────────── │
│  bg: linear-50             │ bg: linear-950               │
│  card: white               │ card: linear-900              │

│  text: linear-900           │ text: linear-50               │
│  text-secondary: linear-500  │ text-secondary: linear-400    │
└─────────────────────────────────────────────────────────────┘

  - 使用 Tailwind dark: 前缀切换
  - 保持色彩对比度 ≥ 4.5:1
  - 避免纯黑 (#000) 或纯白 (#fff)
```
